QE Playbook Assistant

Powered by GitHub Models — free for any GitHub account

This tool uses GitHub Models (gpt-4o) to help you capture QE issues and build a living playbook. To get started, you need a free GitHub personal access token — no payment method required.

1
Open GitHub token settings

Go to github.com → Settings → Developer settings → Personal access tokens → Tokens (classic)

Or visit directly: github.com/settings/tokens/new

2
Generate a new token

Give it any name (e.g. qe-playbook). Set expiration to your preference. No special scopes are required — leave all boxes unchecked.

3
Paste it below and click Connect

Your token is stored locally in secrets.json and sent only to models.inference.ai.azure.com (GitHub's API). It never leaves your machine for any other purpose.
